初步介绍此Demo功能及应用技术：
简介：
 根据WPF基础知识，个人独自制作的演示Demo，用时十天左右。
参考：
 MSDN资料、《WPF编程宝典》、各前辈们博客、基础示例https://github.com/Microsoft/WPF-Samples
组织：
 1. 在主Win窗口左侧为菜单，包含多个可展示的页面Page；右侧为展示区Frame，显示选择页面内容。
 2. 菜单中暂时只添加了PhotoDemo（有关WPF绘图）、DataBinding（WPF数据绑定）、3DDemo（WPF3D）。
实现功能：
 1. PhotoDemo实现打开选择目录，批量添加图片，可对图片进行编辑、旋转缩放平移查看。
 2. DataBinding实现导入XML数据文件，导入数据存储到SqlServer数据库；使用DataGrid展示、编辑数据、对数据增删改并保存结果到数据库。
 3. 3DDemo实现三维场景搭建，变换摄像头实现3D漫游（显示为固定焦点旋转场景+移动焦点旋转场景）。
应用技术：
 1. WPF基础（XAML、属性、资源、样式模板、事件命令、控件）、图形（画笔、图像、形状、变换）
 2. 数据（数据绑定）、数据库基本操作
 3. 三维图形（搭建、变换、动画）、线程处理
